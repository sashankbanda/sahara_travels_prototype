# Sahara Admin Dashboard

## Project Info

**Description**: A premium admin dashboard for Sahara Tours & Travels, featuring package management, enquiry tracking, payment monitoring, and user administration.

## Getting Started

### Prerequisites

Ensure you have Node.js & npm installed.

### Installation

1.  Clone the repository:
    ```sh
    git clone <YOUR_GIT_URL>
    ```

2.  Navigate to the project directory:
    ```sh
    cd sahara-admin-hub
    ```

3.  Install dependencies:
    ```sh
    npm install
    ```

4.  Start the development server:
    ```sh
    npm run dev
    ```

## Development

This project is built using:
- **Vite**: Next Generation Frontend Tooling
- **React**: A JavaScript library for building user interfaces
- **TypeScript**: Typed JavaScript at Any Scale
- **Shadcn UI**: Reusable components built with Radix UI and Tailwind CSS
- **Tailwind CSS**: A utility-first CSS framework
- **Zustand**: A small, fast and scalable bearbones state-management solution

## Deployment

To build the application for production:

```sh
npm run build
```

The build artifacts will be stored in the `dist/` directory.
